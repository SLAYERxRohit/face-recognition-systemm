Meet Viesta

The Next-Gen Face & Age Recognition App

Experience high-performance facial detection, age estimation, and gender classification powered by an end-to-end Machine Learning and Deep Learning pipeline. Viesta locates faces, extracts characteristics, and predicts attributes in real-time.

Launch Analyzer How It Works

Haar Cascades

Locates and crops faces from raw inputs in real-time. Leverages OpenCV's pre-trained Haar Cascades to find human eyes and facial patterns instantly.

PCA & Eigenfaces

Reduces dimensionality and extracts vital features by projecting facial pixels into an eigenspace, mapping raw photos to visual "Eigenfaces".

Gender SVM

Classifies gender features using Support Vector Machine (SVM) algorithms. Computes hyperplane projections to output gender predictions with probability scores.

Age estimation

Predicts age brackets using a deep Caffe convolutional neural network (CNN) model, mapping facial dimensions to 8 separate age groupings.
